WELCOME TO JAVASCRIPT

JavaScript is a object based scripting language which is used to make websites more dynamic and attractive.

ADVANTAGES

1. Lesser server interactions
2. Immediate feedback to user
3. Increased interactivity
4. Richer interface
5. Event driven programming
6. GUI features

DISADVANTAGES

1. Client side JavaScript does not allow the reading and writing of files. This has been kept for security reason.
2. JavaScript cannot be used for networking applications because there is no such support available.
3. JavaScript does not have any multithreading or multiprocessing abilities.

APPLICATIONS

1. Web applications
2. Web server
3. Native mobile applications
4. Hybrid mobile applications

VARIABLES

Variables are used to hold data in a system memory which can later be used by a program.

DATA-TYPES

Data types allow the programmer to determine the data and the valid operation on data and the valid operation on data available in the memory.

Diiferent types of data types in JavaScript are:

1. Numbers
2. Strings
3. Boolean
4. Undefined

LOOPING STATEMENT

Looping statement or iterating statementsare used to execute the same block of code a specified number of times.
Diiferent types of looping statements are:

1. While loop
2. For loop
3. Do while loop

FUNCTIONS

A function is a named block of code which is used to wrap a few statements that should be executed one after other.

OBJECTS

Object is a entity and is the fundamental of JavaScript.
Objects are variables too but objects can contain many values.

ARRAYS

Arrays is an ordered collection of values that is grouped together by a single variable name.
An array is an collection of like values called elements - such as an array of colors, array of strings, an array of images.
Each element of array is accesssed with an index values enclosed in square brackets.
